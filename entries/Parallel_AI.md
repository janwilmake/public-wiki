# Parallel AI

**Parallel AI** is an AI infrastructure company focused on context engineering and tool integration for large language models. [[Jan_Wilmake]] worked as a DevRel contractor at Parallel AI from approximately August to November 2025, focusing on Model Context Protocol (MCP) integrations, context tooling, and Cloudflare-based infrastructure.

## Relationship with Jan Wilmake

During his tenure at Parallel AI, Wilmake contributed to:
- **MCP integrations** — extending the Model Context Protocol ecosystem
- **Context tooling** — developing systems for managing and optimizing context windows in LLM applications
- **Cloudflare infrastructure** — leveraging Cloudflare Workers for scalable AI tool deployment

This work aligned with Wilmake's broader interests in [[Agent_Friendly_Web]] and context engineering, providing commercial experience with enterprise-scale AI infrastructure.

## Technical Alignment

Parallel AI's focus on MCP and context engineering intersected directly with Wilmake's independent projects:
- **[[ContextArea]]** — web-based context engineering tool developed around the same time
- **[[uithub]]** — GitHub repository context viewer

The experience at Parallel AI informed Wilmake's subsequent work on [[Browser_MCP]] and other MCP-related projects in 2026.
