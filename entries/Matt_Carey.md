# Matt Carey

**Matt Carey** ([@mattzcarey](https://x.com/mattzcarey)) is a software engineer at **Cloudflare** who specialises in Durable Objects, MCP infrastructure, and AI agents. He is a recurring collaborator and mutual in [[Jan_Wilmake]]'s network, regularly engaging on topics of Cloudflare Workers, MCP, and agent architecture.

## Role at Cloudflare

Matt works on the developer experience and infrastructure for Cloudflare's agent platform, including work on **Durable Objects** and **Code Mode** for MCP — a Cloudflare-specific MCP feature that allows agents to execute code.

## Connection to Jan

Jan has interacted with Matt extensively on X around:
- MCP design and protocol questions
- Cloudflare Durable Objects patterns and agent architecture
- Code Mode and its implications for OpenAPI-based MCPs
- OAuth in MCP clients

Jan considers Matt part of his core X network, listing him alongside Sam Goodwin, Ben Allfree, and EastlondonDev as people he values.
