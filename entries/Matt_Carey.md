# Matt Carey

**Matt Carey** ([@mattzcarey](https://x.com/mattzcarey)) is a software engineer at **Cloudflare** who specialises in Durable Objects, MCP infrastructure, and AI agents. He is a recurring collaborator and mutual in Jan Wilmake's network, regularly engaging on topics of Cloudflare Workers, MCP, and agent architecture.[^x-posts-2025-10-06]

## Role at Cloudflare

Matt works on the developer experience and infrastructure for Cloudflare's agent platform, including work on **Durable Objects** (frequently posting his appreciation for the technology). He posted about **Code Mode** for MCP — a Cloudflare-specific MCP feature that allows agents to execute code — which Jan found inspiring for his `openapisearch.com` work.[^x-posts-2026-02-24]

## Connection to Jan

Jan has interacted with Matt extensively on X around:
- MCP design and protocol questions
- Cloudflare Durable Objects patterns and agent architecture
- Code Mode and its implications for OpenAPI-based MCPs
- OAuth in MCP clients (raising issues with Cursor's MCP implementation)

When Matt noticed that Jan's `quickog.com` service had a mobile overflow display bug, he filed the feedback directly on X.[^x-posts-2025-12-21]

Jan considers Matt part of his core X network, listing him alongside Sam Goodwin, Ben Allfree, and EastlondonDev as people he values.[^x-posts-2025-10-06]
