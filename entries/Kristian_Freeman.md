# Kristian Freeman

![Kristian Freeman](https://cf-assets.www.cloudflare.com/zkvhlag99gkb/3VUYoHwDSenpoOr2qQ5rQe/5aae862918663bc3e682ede2eb385c22/kristian.jpg)

**Kristian Freeman** is an American software engineer and developer advocate based in Austin (San Marcos), Texas. He works as **Engineering Manager on Cloudflare's Developer Relations team**, focused on developer tooling, AI products, and the Cloudflare Workers ecosystem. He is active on X as [@kristianf_](https://x.com/kristianf_) and on GitHub as [kristianfreeman](https://github.com/kristianfreeman).[^github]

## Career

Freeman started his career writing Ruby and Ruby on Rails applications before moving to JavaScript and React development. He has been an independent educator, running a blog, a YouTube channel (**7.dev**), and the **Byteconf** conference series before joining Cloudflare.[^mesh]

Since July 2022, Freeman has been at **Cloudflare** in developer advocacy and engineering management roles. His work focuses on:

- Writing and producing foundational content for Cloudflare's developer products (Workers, Pages, D1, AI)
- Building example applications and templates for the Workers ecosystem
- Tooling integrations (Algolia, Hugo, GitHub Actions, Wrangler)
- AI agent demos using the Cloudflare Agents SDK

He also runs **Freeman Labs** (since January 2022), a software consulting, staffing, and technical training agency working with companies ranging from startups to Fortune 500.[^mesh]

Outside engineering, Freeman is a **techno DJ and music producer**.[^github]
