# getpromind

**getpromind** (`getpromind.com`) is a personal digital memory aggregation tool developed by [[Jan_Wilmake]]. It stores structured data from multiple sources to serve as a personal knowledge base and context layer for AI agents.

## Features

- **X bookmarks** — stores and indexes bookmarked tweets
- **Browser history** — records and makes searchable web browsing history
- **GitHub repos** — starred and owned repositories

Jan described it in March 2026 when asked about an alternative to screenshot-based memory tools: *"I made http://getpromind.com — it stores more structured data: x bookmarks, browser history, and github repos starred and owned."*

## Context

getpromind fits into Jan's broader interest in personal knowledge infrastructure and the [[socialdatapod]] concept. Rather than screen-scraping or screenshot analysis, it focuses on structured signals from platforms the user already uses.

In February 2026, Jan referenced a related tool at `history.wilmake.com` — a service for giving AI agents access to browsing history. This appears to be either a sibling project to getpromind or an earlier iteration, providing the browsing history component as a standalone service.
